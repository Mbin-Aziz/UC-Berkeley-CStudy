## 📊 Course Topics Covered

- **Functions & Expressions** - First-class functions, lambdas, higher-order functions
- **Data Abstraction** - Lists, tuples, dictionaries, sequences
- **Object-Oriented Programming** - Classes, inheritance, polymorphism
- **Iterators & Generators** - Lazy evaluation, custom iterators
- **Functional Programming** - Map, filter, reduce, comprehensions
- **Scheme Programming Language** - Functional programming paradigm
- **Interpreters** - Building custom language interpreters
- **SQL Databases** - Queries, joins, aggregations, data manipulation
